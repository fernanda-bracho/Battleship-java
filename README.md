# Battleship Game (Java)

A console-based Battleship game developed in Java, applying object-oriented programming principles and game logic.

Features
- Random ship placement
- Hit and miss detection
- Multiple ships with different states
- Interactive user input
- Visual board display (hits and misses)

 How to Play
Enter coordinates in the format:
a0, b3, c6

The goal is to sink all ships in the least number of attempts.

- Object-Oriented Programming (OOP)
- ArrayList for dynamic data handling
- 2D arrays for board visualization
- Control flow and game loop logic

 Technologies
- Java

Future Improvements
- Input validation
- Smarter targeting system (AI)
- Graphical user interface (GUI)
